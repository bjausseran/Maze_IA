# Maze_IA

## Laravel API :

### Project set up :

	1) Open Laragon terminal in Laravel project folder and run : 
		```
		composer install
		```

	2) Fill up .env
		```
		DB_CONNECTION=mysql
		DB_HOST=127.0.0.1
		DB_PORT=3306
		DB_DATABASE=maze_db
		DB_USERNAME=root
		DB_PASSWORD=
		```

	3) Create DB : maze_bd

	4) Open Laragon terminal in Laravel project folder and run : 
		```
		php artisan migrate:fresh --seed
		```
